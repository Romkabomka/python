# Python Basics — Конспект

## Ключевая идея Python

Самое важное в Python — **объекты**.

Python — **объектно-ориентированный язык программирования**. Главная идея Python заключается в том, что **все сущности являются объектами**.

---

## Объекты и классы

* **Объект** — экземпляр класса
* **Класс** — шаблон (прототип) для создания объектов
* На основе одного класса можно создать множество объектов
* У каждого объекта есть **атрибуты**
* Если атрибут — функция, он называется **методом**

## Пользовательские функции


* параметры передаются в круглых скобках

## Отступы в Python

Отступы имеют **ключевое значение**.

Корректно:

```python
def print_name_function(name):
    print(name)
```

Некорректно:

```python
def print_name_function(name):
print(name)
```

## PEP 8

**PEP 8 (Python Enhancement Proposal)** — документ с рекомендациями по стилю кода.

[https://peps.python.org/pep-0008/](https://peps.python.org/pep-0008/)

Основные правила:

1. Использовать пробелы, а не Tab
2. Максимальная длина строки — 79 символов
3. Функции и классы отделяются двумя пустыми строками
4. Импорты — на отдельных строках
5. Inline-комментарии отделяются минимум двумя пробелами

## Комментарии

Python игнорирует комментарии.

```python
# Однострочный комментарий

print(10)  # комментарий в конце строки
```

Многострочные комментарии обычно оформляются как несколько однострочных.

## Выражения и инструкции

**Выражения** возвращают значение:

```python
5 + 3
'a' + 'b'
a > b
```

**Инструкции (statements)** выполняют действия:

```python
my_name = 'Roman'

if my_name:
    print(my_name)
```

Импорт модуля:

```python
import datetime

```

## Переменные

Переменные дают доступ к значениям и содержат **ссылку на объект**.

### Именование (PEP 8)

* `snake_case` — переменные, функции, модули
* `PascalCase` — классы
* `DB_PASSWORD` — константы

Имена должны быть **осмысленными**.

## Динамическая типизация

Python — язык с динамической типизацией.

```python
fruit = 'banana'
fruit = 'apple'
fruit = 'kiwi'
print(fruit)  # kiwi
```

Тип переменной определяется типом присвоенного значения.

---

## Статическая vs динамическая типизация

Пример статической типизации (Java-подобный):

```java
int b = 10;
b = "xyz"; // Error
```

Python:

```python
a = 10
a = True
a = 'Roman'
a = None
```

## Правила работы с переменными

1. Использовать осмысленные имена
2. Для переменных — существительные (`name`, `photos`)
3. Для функций — глаголы (`get_data`, `create_user`)
